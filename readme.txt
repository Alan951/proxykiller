Compatible solo con python v3.x

Antes de empezar...
1. En esta carpeta, ejecuta el comando python3 -m pip install -r requeriments.txt (python puede llamarse python3 o python simplemente)
2. Ejecuta python3 proxykiller.py
3. Presiona enter para deshabilitar manualmente proxy y scripts, cada minuto la tarea es ejecutada en background. 
	Se deshabilita tanto el proxy como los scripts.
4. Ahora si, ponte a jalar.
